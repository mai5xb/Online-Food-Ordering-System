# Online-Food-Ordering-System
The objective of this project is to create a simulated Online Food Ordering System similar to apps and will design and implement this system using Java applying OOP principles and it  supports customers, restaurant owners (admin), menu management, cart, orders, payments, and file saving.

# Features
User Roles
Customer
Browse menu

Add items to cart

View cart

Place orders

Choose payment method (Cash / Card)

View order history

Cancel orders

Restaurant Owner (Admin)
View full menu

Add new food items

Remove food items

View all orders

Update order status

# How to Run
Open the project in any Java IDE (IntelliJ, Eclipse, NetBeans).

Make sure all .java files are inside the same package or folder.

Run main.java.

Login as customer or admin.

Start ordering or managing the menu.
